## Step by step

- Environtment setup: <p> 
https://reactnative.dev/docs/environment-setup
- Clone the repository: <p> 
`git clone https://github.com/AlexisFabianCarranza/React-Native-Movie-Finder.git`
- Install dependencies: <p> 
`yarn`
- Run project: 
`npm run android`
`npm run ios`
